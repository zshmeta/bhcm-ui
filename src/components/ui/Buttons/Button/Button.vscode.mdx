# Button

This file is meant to preview well in VS Code MDX preview extensions without running Storybook/Vite.

## Basic usage

```tsx
import Button from './Button'

export function Example() {
  return <Button kind="primary">Primary</Button>
}
```

## Kinds

```tsx
import Button, { BUTTON_KINDS } from './Button'

export function Kinds() {
  return (
    <div style={{ display: 'flex', flexWrap: 'wrap', gap: 12 }}>
      {BUTTON_KINDS.map(kind => (
        <Button key={kind} kind={kind} onClick={() => {}}>
          {kind}
        </Button>
      ))}
    </div>
  )
}
```

## Notes

- VS Code MDX preview extensions typically do not execute React imports/JSX; they render MDX as documentation.
- Use [src/components/ui/Buttons/Button/Button.mdx](src/components/ui/Buttons/Button/Button.mdx) for Storybook Docs rendering.
